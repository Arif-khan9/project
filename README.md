
This project is a frontend implementation created as part of the
*Percipience Labs Senior Frontend Developer Assignment*.

The application is built using *React, **TypeScript, **Vite*, and
*Tailwind CSS*, with a focus on reusable components, responsive
layouts, and clean code architecture.

## Tech Stack

-   React 18
-   TypeScript
-   Vite
-   Tailwind CSS
-   Functional Components
-   React Hooks

## Assumptions Made

-   The implementation is based solely on the provided design reference.
-   Since no backend or API requirements were provided, all content is
    static.
-   The focus was on building reusable, modular React components.
-   The layout is responsive across desktop, tablet, and mobile screen
    sizes.

## Additional Improvements
